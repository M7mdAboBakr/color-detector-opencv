# Color Detection with OpenCV

Real-time color detection using webcam feed. Detects specified colors and draws bounding boxes around them.

## Features

- Real-time color detection from webcam
- Bounding box visualization
- Customizable target colors

## Available Colors

- `red`, `blue`,`green`, `yellow`
- `orange`, `purple`, `pink`, `brown`
- `black`, `white`, `gray`, `cyan`
- `magenta`, `lime`, `teal`, `navy`
- And many more CSS color names!
